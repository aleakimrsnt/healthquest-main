# healthquest

> A mobile application built with React Native and powered by Firebase, designed to promote healthier dietary habits through gamification and community engagement.

## Features

- User authentication (Login, Sign Up, Forgot Password)
- Profile management and avatar customization
- Gamified health tracking system
- Daily food logging (Breakfast, Lunch, Dinner)
- Water intake tracking and completion tasks
- Exercise tracking and activity completion
- Leaderboards for user engagement and competition
- Community interaction features
- Rewards and achievement system (Awards screen)
- Store and Food Bank system for in-app resources
- Settings and user preferences management

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- Expo CLI (recommended)
- Firebase account
- Expo Go app for mobile testing

---

## Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/healthquest-main.git
cd healthquest
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the project
```bash
npx expo start
```

### 4. Run the app
- Scan QR code using Expo Go
  - If using Expo Go, install the compatible SDK 49 version here: https://expo.dev/go?sdkVersion=49&platform=android&device=true  
  - This project was developed using an older Expo SDK version and has not been updated to the latest release.
- Or run on emulator

---

## Usage

1. Create an account or log in  
2. Set up your profile and avatar  
3. Track daily meals (Breakfast, Lunch, Dinner)  
4. Complete daily tasks such as water intake and exercise  
5. Earn rewards and climb the leaderboard  
6. Engage with the community and explore Food Bank and Store features
